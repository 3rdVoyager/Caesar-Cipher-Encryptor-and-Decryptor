CaesarCracker

A Caesar cipher encryption and decryption tool written in Python. Supports standard encryption and decryption with a known shift, as well as a brute-force decryption mode for messages with an unknown shift.

Features
Encrypt messages using a shift value of 1–25
Decrypt messages with a known shift
Brute-force decrypt messages with an unknown shift, filtering results by plausibility using vowel/consonant pattern analysis and common English words
Preserves spaces, punctuation, and numbers
Simple interactive command-line interface

How to Run
Make sure you have Python 3 installed, then run:
python caesarcrack.py
